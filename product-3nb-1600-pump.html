3NB-1600 Triplex Mud Pump

API Standard Drilling Mud Pump - 1600HP (1176KW)

The 3NB-1600 is a triplex single-acting piston pump designed according to API standards for oilfield drilling operations. With 1600HP (1176KW) input power and optimized parameters, this pump features increased stroke length (305mm) and displacement, while reasonably lowering stroke speed to improve service life of expendable parts. Manufactured according to SY5138 standard of Petroleum Industry Ministry and tested accordingly.
