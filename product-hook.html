Hook (Rotary Hook)

Professional Drilling Equipment

The Rotary Hook is designed for suspending loads during drilling operations, featuring a main hook for heavy loads and an auxiliary hook for lighter operations.

Key Features

  • Hydraulic buffering - Protects drill pipe while breaking out
  • Security facility - Reliable latch in hook mouth
  • Quality control - Nondestructive flaw detection
  • Certification - API Spec 8A regulations
