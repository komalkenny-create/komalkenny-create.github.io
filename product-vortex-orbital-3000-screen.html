Vortex Orbital 3000 Shaker Screen

Premium Replacement Solids Control Equipment

The Vortex Orbital 3000 shaker screen is a high-performance replacement screen designed for Vortex series shale shakers. Featuring rectangular perforated wear plate and SS 304/316 wire mesh cloth construction, these screens deliver superior solids control efficiency and extended service life for demanding drilling operations.

Manufactured with premium stainless steel 304/316/316L materials and Q235 steel frame, compliant with API RP 13C standards, Vortex Orbital 3000 screens are available in API 20 to API 325 mesh designations. Compatible with Vortex BSBS Linear Power Side-By-Side Shaker, Vortex B DHC Double Header Mud Cleaner, and Vortex B CD VGS™ Linear Cutting Drier systems.
