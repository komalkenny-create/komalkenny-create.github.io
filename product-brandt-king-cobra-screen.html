Brandt King Cobra/Hybrid Shaker Screen

Premium Replacement Solids Control Equipment

The Brandt King Cobra/Hybrid shaker screen is a high-performance replacement screen designed for Brandt King Cobra and Hybrid shale shakers. Featuring SS 304/316 wire mesh cloth with either ribbed back supporting frame (steel) or composite supporting frame (PT frame) construction, these screens deliver superior solids control efficiency and extended service life.

Manufactured with premium stainless steel 304/316/316L materials and Q235 steel/composite frame, compliant with API RP 13C standards, King Cobra screens are available in API 20 to API 325 mesh designations. With a working life of 400-450 hours and 1-year warranty, these screens provide reliable performance for demanding drilling operations.

Product Features

  • Premium Materials: Stainless steel 304/316/316L wire mesh for superior corrosion resistance
  • Dual Frame Options: Q235 steel frame or composite frame (PT frame) available
  • Ribbed Back Design: Supporting frame with ribbed back for improved structural integrity
  • SS Wire Mesh Cloth: Advanced screen cloth technology for optimal separation
  • PT Frame Technology: Composite supporting frame for enhanced performance
  • API RP 13C Compliant: Meets industry standards for drilling fluid solids control equipment
  • Extended Service Life: 400-450 hours working life for reduced replacement costs
  • 1-Year Warranty: Quality assurance and manufacturer support
  • Multiple Series Options: XR, XL series available for various applications
  • Wide API Range: API 20 to API 325 mesh designations for diverse solids control needs
